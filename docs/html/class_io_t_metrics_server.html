Detailed Description

Namespace aliases for OpenTelemetry metrics API and SDK.

IoTMetricsServer provides an HTTP API for ingesting and exporting OpenTelemetry metrics.

This server supports Counter, UpDownCounter, Histogram, and Gauge instruments, and exposes Prometheus-compatible endpoints.
